Simple IT Ticketing app, vibecoded with Claude and deployed in Vercel with Supabase as database
Features:
- Row level security and email authentication with Supabase  
- Gemini AI used to give categorization (type of ticket and priority) recommendation
- Company asset that can be added to ticket
- In ticket: status, due date, made by, assign to, SLA Due, categoty, priority, and comment section

The website can be accessed here:
https://it-ticketing-asset-management-system.vercel.app/dashboard/tickets
